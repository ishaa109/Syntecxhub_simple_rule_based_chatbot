# Simple Rule-Based Chatbot

## Project Overview

This project implements a simple Rule-Based Chatbot using Python. The chatbot uses pattern matching techniques to identify user intents and provide predefined responses.

The chatbot supports:
- Greetings
- Help requests
- Small talk conversations
- Domain-specific questions using a knowledge base
- Conversation history logging

## Features

- Pattern matching based chatbot
- Interactive console interface
- Knowledge base for predefined questions
- Chat history logging
- Easy to customize and extend

## Technologies Used

- Python
- File Handling
- Rule-Based AI Concepts
